# Virtual Experimental World

Name: **NeuroTwin virtual experimental world**

Turn brain-system hypotheses into executable dry experiments with explicit evidence gates and learning memory.

## Layers

| Layer | Role | Artifact |
| --- | --- | --- |
| Problem Space | Represent task context, target phenotype, intervention objective and constraints. | Hypothesis + objective function |
| Surrogate Simulator | Predict brain dynamics and run controllable perturbations before costly validation. | BOLD, FC and virtual EC response |
| Evidence Gates | Score fidelity, robustness, biological plausibility and validation cost. | Trace-backed metrics |
| Learning Memory | Retain top choices, weak actions and uncertainty targets for the next DSVL cycle. | Acquisition portfolio |

## Why It Matters

The key AI4S value is the ability to run many low-cost virtual trials before selecting expensive data collection or human validation. The virtual world keeps hypotheses, simulation outputs, validation gates and learning memory in one traceable loop.

## Expansion Interfaces

- Public fMRI datasets for task-specific or resting-state validation.
- Behavioral endpoints for objective calibration.
- Molecular, protein or gene model outputs as upstream mechanism hypotheses.
- Lab feedback from neurofeedback, stimulation planning, organoid readout or expert review.
