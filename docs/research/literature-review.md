### 论文 1

**题目**: Mapping effective connectivity by virtually perturbing a surrogate brain
**文件**: `paper/NeuralPerturbationalInference.pdf`

#### 研究问题

作者关注的问题是：如何在**不进行真实侵入式刺激**的前提下，从脑成像数据中推断脑区之间的**有效连接**（effective connectivity, EC），即带有方向性和因果意义的连接关系。

#### 核心思想

作者提出了 **Neural Perturbational Inference (NPI)** 框架。其核心思路是：

1. 先训练一个人工神经网络，使其能够预测下一时刻的脑活动。
2. 把这个训练好的网络视为“替代大脑”或“代理脑”。
3. 在这个代理脑中对某个脑区施加虚拟扰动。
4. 观察其他脑区预测输出的变化，从而推断该脑区对其他脑区的因果影响。

换句话说，NPI 本质上是在做一种“**先学动力学，再做虚拟刺激实验**”。

#### 方法流程

1. 输入连续多个时间步的脑信号，预测下一个时间步。
2. 用预测生成的信号重建功能连接（FC），若模型生成的 FC 与真实 FC 高度一致，说明该代理模型学到了较真实的脑区交互规律。
3. 固定模型后，逐个扰动脑区节点，计算扰动前后其他节点输出的变化。
4. 汇总得到全脑的 EC 矩阵。

论文中使用的代理模型主要是 MLP，输入为前三个时间步的信号，输出为下一时间步信号。作者也测试了 CNN、RNN 和 VAR 等替代模型。

#### 主要结果

1. 在带有已知“真实 EC”的模拟数据上，NPI 能较准确恢复因果连接，性能优于 Granger causality 和 DCM。
2. 在 HCP 的 800 名被试 rs-fMRI 数据上，模型生成的 FC 与真实 FC 相关性达到 `r = 0.97`，说明替代脑对整体脑动力学拟合较好。
3. 推断得到的人脑有效脑连接图谱（effective brain connectome, EBC）具有稳定的网络结构特征：
   - 兴奋性连接更强且多为网络内连接
   - 抑制性连接更多出现在网络间和双半球之间
4. 跨图谱分辨率、跨会话、跨数据集（HCP 与 ABCD）结果具有较好稳健性。
5. 与真实电刺激传播数据（CCEP）相比，NPI 推断的 EC 与刺激传播模式更一致，优于传统 FC。

#### 论文贡献

1. 提出了一种从相关性走向因果推断的非侵入式框架。
2. 不只给出“是否相关”，还能给出连接方向、强度和兴奋/抑制属性。
3. 为神经调控、刺激靶点选择和脑疾病机制分析提供了可用工具。

#### 局限性

1. 高度依赖代理模型是否真正学到了脑动力学。
2. 需要较大规模、高质量数据。
3. 结果是“数据驱动因果近似”，并不等价于真实生物刺激实验中的绝对因果真值。

#### 对本课题的启发

这篇论文对你的课题非常关键，因为它告诉我们：**数字孪生脑不一定先从完整生物物理机制入手，也可以先训练一个能够复现脑动力学的代理模型，再在模型中做虚拟实验。**

它直接启发了三个方向：

1. 可以把“数字孪生脑”理解为一个可扰动、可观测、可验证的代理脑系统。
2. 可以把功能连接分析从静态 FC 进一步推进到带方向性的 EC。
3. 在后续课题中，我们不仅可以做状态预测，还可以做“虚拟刺激”和“功能仿真”。

### 论文 2

**题目**: Digital Twin Brain: Generating Multitask Behavior from Connectomes for Personalized Therapy
**文件**: `paper/bmef.0231.pdf`

#### 研究问题

作者试图回答的问题是：能否从个体的脑连接组出发，构建一个能够同时生成**行为、脑活动和治疗调制效果**的数字孪生脑模型，进而服务于个体化精神疾病治疗。

#### 核心思想

作者提出了一个两部分结构的数字孪生脑系统：

1. **Hypernetwork**
   输入个体的静息态功能连接矩阵（rsFCM），输出一个个体化的主网络参数。
2. **Main network**
   是一个 RNN，接收任务刺激和任务条件输入，输出行为反应与 BOLD 信号。

核心逻辑是：
**用连接组生成“这个人专属”的神经网络参数，再让这个专属网络去模拟该人在不同任务中的脑和行为表现。**

#### 方法流程

1. 从静息态 fMRI 中提取个体 rsFCM，作为个体脑连接特征。
2. Hypernetwork 根据 rsFCM 生成 Main network 的全部参数。
3. Main network 接收任务输入，联合预测：
   - 下一时刻行为输出
   - 下一时刻 BOLD 信号
4. 在此基础上，再利用梯度反传寻找“如果想改变认知/情绪功能，应该如何修改连接组”。
5. 最后做个体化虚拟干预，预测不同个体对干预的反应差异。

#### 数据与任务

1. 数据来自 TCP 数据集，共 228 名被试。
2. 任务覆盖两个功能域：
   - Emotional Faces 任务，对应情绪/负性效价系统
   - Stroop 任务，对应认知控制/加工速度
3. 采用静息态连接组作为个体特征，同时用任务态行为和 BOLD 作为建模目标。

#### 主要结果

1. 行为预测效果较好：
   - Emotional Faces 任务选择预测准确率约 `0.94`
   - Stroop 任务选择预测准确率约 `0.90`
2. 反应时预测与真实值相关较高：
   - Emotional Faces: `r = 0.90`
   - Stroop: `r = 0.85`
3. 预测 BOLD 与真实任务相关激活模式较一致，GLM t 统计相关达到 `r = 0.84`
4. 利用模型可定义“情绪反应”和“加工速度”两个功能指标，并反向寻找最有效的连接调制方向。
5. 虚拟干预结果显示：
   - 可降低杏仁核情绪反应
   - 可缩短反应时
   - 干预效果存在明显个体差异

#### 论文贡献

1. 这篇论文比一般“连接预测行为”更进一步，真正做到了“连接组 -> 动态脑活动/行为生成”。
2. 它不是只做单任务预测，而是尝试在统一框架下覆盖多个功能域。
3. 它将数字孪生脑与个体化干预设计联系起来，具备较强应用导向。

#### 局限性

1. 任务数量仍然较少，只覆盖两个功能域。
2. BOLD 被插值到更高时间分辨率，这一处理虽然做了稳健性检验，但仍值得谨慎理解。
3. 从“连接改变”到“真实可实施治疗”之间还有很大距离。
4. 样本量对真正的临床泛化仍然有限。

#### 对本课题的启发

这篇论文和你的题目几乎是直接相关的，因为它已经非常接近“基于 MRI 的数字孪生脑建模与功能仿真”。

它给出了一条非常清晰的研究路线：

1. 用 MRI/fMRI 提取个体连接组特征；
2. 用深度学习模型建立个体化数字孪生脑；
3. 让模型生成任务相关行为和脑活动；
4. 在模型内部做虚拟干预和功能仿真；
5. 研究个体差异与潜在治疗价值。

### 两篇论文的关系

这两篇论文都不是在做传统意义上的简单分类，而是在做“**可模拟、可解释、可干预的代理脑建模**”。

1. 第一篇更偏**连接推断**：
   重点是从脑信号中恢复 EC，强调因果结构和脑网络传播机制。

2. 第二篇更偏**数字孪生生成**：
   重点是从连接组出发生成行为与脑活动，并进一步做个体化虚拟干预。

3. 两者可以形成组合路线：
   - 先用类似 NPI 的思路理解脑区因果连接结构
   - 再用数字孪生框架进行任务态功能仿真与干预模拟

### 当前认识

结合这两篇论文，可以把本课题初步理解为以下几个层次：

1. **数据层**：采集和处理 fMRI/rs-fMRI 数据，提取脑区时间序列与连接特征。
2. **网络层**：构建 FC、动态 FC，进一步尝试 EC。
3. **模型层**：训练能够复现脑动力学或任务行为的代理模型。
4. **孪生层**：将代理模型个体化，形成数字孪生脑。
5. **仿真层**：在模型中进行任务响应预测、脑区协同可视化与虚拟干预。

这也意味着，你的课题未来很可能不是只做“脑连接分析”，而是朝着“**个体化脑网络建模 + 功能仿真 + 虚拟干预原型**”的方向发展。
